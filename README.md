Winterrr is a simple, static front-end web project created to showcase winter-themed activities, sports, and gear. It features a clean layout focused on outdoor winter fun, ranging from competitive skiing to cozy night camping.

🛠️ Built With
HTML5
CSS3 (Custom desktop styles)
JavaScript

📂 Repository Structure
index.html – The core structure of the landing page.
desktop.css – Main stylesheet for layout and design.
script.js – Basic client-side logic.
*.png / *.jpg – Asset files for winter sports, equipment, and achievement icons.
